# Data

## Sleipner Reference Model

### Overview
The Sleipner Reference Model represents the Utsira Formation in the North Sea.

**Specifications:**
- Total cells: 1,986,176
- Permeability (PERMX): 0-2,000 mD
- Porosity (PORO): 0.34-0.36
- Time-lapse layers: 9 (L1-L9)
