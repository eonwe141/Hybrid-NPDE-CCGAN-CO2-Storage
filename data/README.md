# Data

## Sleipner 2019 Benchmark Model

### Overview

This work uses the **Sleipner 2019 Benchmark Model**, a reference dataset from the Sleipner CO2 storage site located off the western coast of Norway. The site has been actively used by Equinor for CO2 storage since 1996.

### Dataset Information

**Official Details:**
- **Title:** Sleipner 2019 Benchmark Model
- **DOI:** [10.11582/2020.00004](https://doi.org/10.11582/2020.00004)
- **URL:** https://co2datashare.org/dataset/sleipner-2019-benchmark-model
- **Provider:** Equinor / CO2DataShare
- **Released:** January 17, 2020
- **License:** SLEIPNER CO2 REFERENCE DATASET LICENSE
- **Access:** Open
- **Period:** January 1994 - December 2010

**Dataset Includes:**
- Simulation grid with petrophysical properties (PERMX, PORO)
- Well data and logs
- Velocity maps
- Individual horizons within storage reservoir
- Time-lapse seismic data

**Contact:** Philip Ringrose, CO2DataShare/CSDC Secretariat
